============
AioneMotor_DSC28034
============

AioneMotor_DSC28034电机驱控一体板使用了中科昊芯HXS320F28034PNT RISC-V DSP芯片，该芯片集成了中科昊芯自主研发的H28x内核。该电机驱控一体板可驱动BLDC，PMSM，交流感应等三相电机。适合有感/无感方波，有感/无感FOC等算法演示。
