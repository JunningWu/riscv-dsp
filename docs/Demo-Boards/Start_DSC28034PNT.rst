============
Start_DSC28034PNT
============

Start_DSC28034湖人开发板使用了中科昊芯HXS320F28034PNT RISC-V DSP芯片，该芯片集成了中科昊芯自主研发的H28x内核。基于昊芯高性能H28x内核的HXS320F28034PNT处理器，主频高达120MHz，扩展资源丰富SCI(UART)、SPI、（I2C）、LIN、eCAN。
